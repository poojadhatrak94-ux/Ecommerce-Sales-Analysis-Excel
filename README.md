# E-Commerce Sales Analysis Dashboard

## Project Overview
Built an interactive Excel dashboard to analyze e-commerce sales performance across regions, categories, customers, and segments.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

## KPIs
- Total Sales: ₹22,61,331.75
- Total Orders: 9,799
- Total Customers: 794
- Categories: 3
- Regions: 4
- Average Sales per Order: ₹230.77

## Key Insights
- Technology is the highest-selling category.
- Consumer segment contributes 51% of total sales.
- West region generated the highest revenue.
- Sean Miller is the top customer.
- Average Sales per Order is ₹230.77.

## Dashboard Features
- Interactive slicers
- Sales by Region
- Sales by Category
- Sales by Segment
- Top 10 Customers Analysis
- KPI Cards

# E-Commerce Sales Analysis Dashboard

## Project Overview
This project analyzes e-commerce sales data using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

## KPIs
- Total Sales
- Total Orders
- Total Customers
- Categories
- Regions
- Average Sales per Order

## Key Insights
- Technology generated highest sales.
- Consumer segment contributed 51% of sales.
- West region generated highest revenue.
- Sean Miller was the top customer.
- Average Sales per Order was ₹230.77.
